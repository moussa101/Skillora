FRONTEND ENGINEER - META

Location: Menlo Park, CA
Team: Facebook Core Product

We're looking for engineers who are passionate about creating seamless, accessible, and performant user experiences.

- 4+ years of programming experience in JavaScript
- 4+ years of experience building user interfaces with React or similar frameworks
- Experience with web technologies including HTML5, CSS3, and responsive design
- Experience with state management solutions (Redux, MobX, Recoil)

Preferred Qualifications:
- Experience with TypeScript
- Experience with GraphQL or RESTful APIs
- Experience with testing frameworks (Jest, React Testing Library, Cypress)

- Experience with performance optimization and debugging

- Experience with mobile development (React Native)

Responsibilities:

- Participate in code reviews and provide constructive feedback

- Stay up-to-date with emerging technologies and industry trends
- Mentor junior engineers and contribute to engineering best practices

Tech Stack:
React, TypeScript, GraphQL, Relay, Jest, Flow
